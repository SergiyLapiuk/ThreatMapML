# ThreatMap

**ThreatMap** — це інтелектуальна система для автоматизованого аналізу неструктурованих текстових повідомлень про повітряні загрози (з OSINT-джерел та Telegram-каналів) та їхньої миттєвої візуалізації на інтерактивній мапі. 

Проєкт розроблено в рамках магістерської дисертації.

## Основні можливості

* **Семантичний аналіз тексту:** Автоматичне видобування сутностей (тип загрози, кількість, точка запуску, напрямок руху) із сирого тексту за допомогою великих мовних моделей (LLM).
* **Геокодування:** Автоматичне перетворення назв населених пунктів на географічні координати за допомогою інтеграції з OpenStreetMap (Nominatim API) із вбудованим механізмом кешування.
* **Візуалізація в реальному часі:** Відображення загроз на карті за допомогою маркерів із розрахунком азимута (bearing) для правильного направлення іконок об'єктів.
* **Багатомодельний бенчмарк:** Вбудований сервіс порівняння (`MultiModelLLMService`) для тестування ефективності різних моделей (MamayLM, Mistral, Zephyr, Qwen, Phi).

## Стек технологій

**Бекенд:**
* **Python 3.10+**
* **FastAPI** — для створення швидкого REST API.
* **PyTorch & Transformers (HuggingFace)** — для завантаження та виконання нейронних мереж (LLM).
* Ядро аналізу: **`INSAIT-Institute/MamayLM-Gemma-3-4B-IT-v1.0`** (оптимізована україномовна модель).

**Фронтенд:**
* **JavaScript (Vanilla)**
* **Leaflet.js** — для рендерингу інтерактивної карти.
